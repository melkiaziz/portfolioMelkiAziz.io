# Portfolio de Melki Mohamed Aziz

Ce projet est un portfolio professionnel en ligne moderne et responsive, conçu pour présenter :
- le parcours académique et professionnel,
- les projets réalisés,
- les compétences techniques,
- les informations de contact.

## Structure
- index.html : structure du site
- styles.css : styles et design responsive
- script.js : interactions simples et formulaire de contact

## Lancement local
Ouvrez simplement le fichier index.html dans un navigateur, ou lancez un serveur local :

```bash
python -m http.server 8000
```

Puis ouvrez http://127.0.0.1:8000/
